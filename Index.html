About SushiSwap

SushiSwap is a community-driven decentralized exchange (DEX) built on Ethereum and other blockchains. It allows users to swap cryptocurrencies directly from their wallets, provide liquidity, earn yield, and participate in governance using the $SUSHI token.

How It Works

SushiSwap uses smart contracts to allow peer-to-peer token swaps without intermediaries. Liquidity providers deposit token pairs into pools and earn a portion of transaction fees. The protocol uses algorithms to manage prices based on supply and demand.
